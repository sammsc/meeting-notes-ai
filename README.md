# meeting-notes-ai
Summarizing meeting transcript.
The transcript format is as follow:

```
Speaker 1 (mm:ss):
speaker 1 words.

Speaker 2 (mm:ss):
speaker 2 words.

Speaker 1 (mm:ss):
speaker 1 words.

Speaker 2 (mm:ss):
speaker 2 words.
```

To start the web UI:
```
streamlit run meeting_notes.py
```
